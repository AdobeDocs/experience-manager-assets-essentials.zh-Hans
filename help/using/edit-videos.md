# 在 [!DNL Assets Essentials] 中编辑视频 {#edit-videos}

Assets 用户可以通过嵌入的 [!DNL Adobe Express] 视频快速操作轻松创建视频内容的变体。 [!DNL Assets Essentials] 中的快速操作功能由 [!DNL Adobe Express] 提供支持，提供了用户友好的视频编辑选项，包括裁切视频、调整视频大小、修剪视频以及将视频转换为 GIF。

要编辑视频，请前往视频详细信息，然后点击[!UICONTROL 编辑视频]。 或者，选择资产后点击详细信息，然后点击右窗格中提供的![剪刀](assets/do-not-localize/cut.svg)图标。 视频编辑完成后，您可以将新视频另存为一个新版本或新资产。

详细了解[预览界面](/help/using/navigate-view.md#preview-assets)。

## 先决条件 {#prerequisites}

有权访问 [!DNL Adobe Express] 和 AEM Assets 中的至少一个环境。 环境可以是 [!DNL Assets as a Cloud Service] 或 [!DNL Assets Essentials] 内的任何存储库。

## 使用 Adobe Express 编辑视频 {#edit-video-using-express}

>[!CONTEXTUALHELP]
>id="assets_express_integration_videos"
>title="用于视频编辑的 Adobe Express 集成"
>abstract="Experience Manager Assets 现在包括由 Adobe Express 支持的简单而直观的视频编辑工具，可提高内容重用率，并加快内容创建速度。 编辑选项包括修剪、裁剪、调整视频大小以及将 MP4 转换为 GIF 文件。"

使用嵌入的 [!DNL Adobe Express] 快速操作功能可以轻松地将视频转换为完美的尺寸和定位。

### 裁切视频 {#crop-video-using-express}

您可以使用嵌入的 [!DNL Adobe Express] 快速操作方法从视频中去除不想要的部分。 为此请执行以下步骤：

1. 选择一个视频，然后点击&#x200B;**[!UICONTROL 编辑]**。
2. 从左窗格中提供的快速操作功能中点击&#x200B;**[!UICONTROL 裁切视频]**。
3. 拖动视频四角上的手柄，创建您需要的裁切效果；或者根据需要从现有的屏幕尺寸中进行选择。
4. 您可以选择将视频静音或取消静音。
5. 单击&#x200B;**[!UICONTROL 应用]**。
   ![使用 Adobe Express 裁切视频](/help/using/assets/adobe-express-crop-video.png)

   裁切的视频可供下载。 您可以将编辑后的资产另存为同一资产的新版本，也可以将其另存为新资产。 ![使用 Adobe Express 保存视频](/help/using/assets/adobe-express-save-video.png)

### 调整视频大小 {#resize-video-using-express}

DAM 中的最终视频内容通常需要调整大小才能分发到特定渠道。 [!DNL Assets Essentials] 可让您轻松调整视频大小，以适合常用社交渠道所需的尺寸，您还可以根据自定义分辨率调整大小。 要使用 [!DNL Assets Essentials] 调整视频大小，请执行以下步骤：

1. 选择一个视频，然后点击&#x200B;**[!UICONTROL 编辑]**。
2. 从左窗格中提供的快速操作功能中点击&#x200B;**[!UICONTROL 调整视频大小]**。
3. 从社交媒体平台的&#x200B;**[!UICONTROL 调整大小]**&#x200B;下拉列表中选择合适的尺寸。 或者，拖动视频四角上的手柄，创建您想要的裁切效果。
4. 如果需要，使用&#x200B;**[!UICONTROL 视频比例]**&#x200B;字段缩放视频。
5. 您可以选择将视频静音或取消静音。
6. 单击&#x200B;**[!UICONTROL 应用]**以应用您的更改。
   ![使用 Adobe Express 调整视频大小](/help/using/assets/adobe-express-resize-video.png)

调整大小后的视频可供下载。 您可以将编辑后的资源另存为同一资源的新版本，也可以将其另存为新资源。

### 修剪视频 {#trim-video-using-express}

如果需要使用较大视频的一个片段，您可以使用“修剪视频”功能来选择和修剪视频的一部分。 执行以下步骤：

1. 选择一个视频，然后点击&#x200B;**[!UICONTROL 编辑]**。
2. 从左窗格中提供的快速操作功能中点击&#x200B;**[!UICONTROL 修剪视频]**。
3. 指定视频的开始和结束时间，以修剪视频的一个特定部分。 或者，拖动视频四角上的手柄，创建您想要的修剪部分。
4. 从&#x200B;**[!UICONTROL 尺寸]**&#x200B;下拉列表中选择合适的尺寸。
5. 您可以选择将视频静音或取消静音。
6. 单击&#x200B;**[!UICONTROL 应用]**以应用您的更改。
   ![使用 Adobe Express 调整视频大小](/help/using/assets/adobe-express-trim-video.png)

修剪好的视频可供下载。 您可以将编辑后的资源另存为同一资源的新版本，也可以将其另存为新资源。

### 将视频转换为 GIF {#convert-mp4-to-gif-using-express}

您可以使用 Adobe Express 将 MP4 视频快速转换为 GIF 格式。 执行以下步骤：

1. 选择一个视频，然后点击&#x200B;**[!UICONTROL 编辑]**。
2. 从左窗格中提供的快速操作功能中点击&#x200B;**[!UICONTROL 转化为 GIF]**。
3. 根据所需的质量选择适当的文件大小。 另外，选择横向、纵向方向或者正方形。
4. 拖动视频四角上的手柄，以创建您想要的裁切效果。
5. 单击&#x200B;**[!UICONTROL 应用]**。

   ![使用 Adobe Express 将视频转换为 GIF](/help/using/assets/adobe-express-convert-video-to-gif.png)

您的视频可以 GIF 格式下载。 您可以将编辑后的资源另存为同一资源的新版本，也可以将其另存为新资源。

## 限制 {#limitations-video-adobe-express}

* 只有 MP4 格式的视频可以进行编辑。

* 受支持的最大源文件为 1 GB。

* 受支持的视频在任何一边都介于 46 个像素和 3840 个像素之间。

* 受支持的 Web 浏览器包括 Google Chrome、Firefox、Safari 和 Edge。

* 此功能无法在 Web 浏览器的隐身模式下打开。
